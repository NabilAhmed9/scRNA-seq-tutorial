## Installation Commands

### Install from CRAN

```r
install.packages(c(
  "Seurat",
  "tidyverse",
  "Matrix",
  "scales",
  "cowplot",
  "patchwork",
  "clustree",
  "RCurl"
))
```

### Install from Bioconductor

```r
if (!require("BiocManager", quietly = TRUE))
  install.packages("BiocManager")

BiocManager::install(c(
  "SingleCellExperiment",
  "ComplexHeatmap"
))
```

### Install from GitHub

```r
if (!require("remotes", quietly = TRUE))
  install.packages("remotes")

# Seurat wrappers for Monocle3 integration
remotes::install_github("satijalab/seurat-wrappers")

# Monocle3
remotes::install_github("cole-trapnell-lab/monocle3")

# CellChat
remotes::install_github("sqjin/CellChat")
```

> **Note for CellChat installation:** CellChat requires NMF which may need additional system dependencies. On Linux:
> ```bash
> sudo apt-get install libopenblas-dev
> ```
> On macOS with Homebrew:
> ```bash
> brew install openblas
> ```

## Input File Types

| File Type | Format | Description |
|---|---|---|
| `barcodes.tsv.gz` | Gzip-compressed TSV | Cell barcode list from CellRanger |
| `features.tsv.gz` | Gzip-compressed TSV | Gene identifier list from CellRanger |
| `matrix.mtx.gz` | Gzip-compressed Matrix Market | Sparse UMI count matrix from CellRanger |

These three files are the standard **10X Genomics CellRanger v3+ output** format, read by `Seurat::Read10X()`.

## Recommended Environment Reproducibility

For full reproducibility, capture your session information after analysis:

```r
writeLines(capture.output(sessionInfo()), "session_info.txt")
```

Or use `renv` for package version locking:

```r
install.packages("renv")
renv::init()      # at start of project
renv::snapshot()  # after installing packages
```

The `renv.lock` file can then be committed to the repository and used by others to restore the exact package versions:

```r
renv::restore()
```

## Known Compatibility Notes

1. **Seurat v4 vs. v5:** The code uses Seurat v5 layer syntax (`LayerData()`, `JoinLayers()`). These functions are not available in Seurat v4. If using v4, replace `LayerData()` with `GetAssayData()` and remove `JoinLayers()` calls.

2. **Monocle3 + SeuratWrappers:** The `as.cell_data_set()` function is only available after loading both `monocle3` and `SeuratWrappers`. Load order matters.

3. **CellChat database:** The code uses `CellChatDB.mouse`. For human data, replace with `CellChatDB.human`.

4. **Mouse gene naming:** The code uses `str_to_title(cc.genes$s.genes)` to convert human gene symbols to mouse title case (e.g., `PCNA` → `Pcna`). This is required because Seurat's built-in `cc.genes` list uses human HGNC symbols.
